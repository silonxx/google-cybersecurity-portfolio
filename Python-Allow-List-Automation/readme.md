# Algorithm for File Updates in Python

A Python algorithm that automates updating an allow list of IP addresses used to control access to restricted content. Reads the allow list from a file, removes any IP addresses flagged on a separate remove list, and writes the updated list back to the file. Demonstrates use of `with` statements, `.read()`, `.split()`, `for` loops, `.remove()`, `.join()`, and `.write()`.
