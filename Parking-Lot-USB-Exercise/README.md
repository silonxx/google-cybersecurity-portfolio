# Parking Lot USB Exercise

A tabletop exercise analyzing the risks of USB baiting attacks, based on a scenario where a USB drive is found in a hospital parking lot. The exercise walks through examining the drive's contents in an isolated virtual environment, thinking through an attacker's perspective, and identifying controls to mitigate USB-based threats.

📄 [View the full exercise](Parking%20lot%20USB%20exercise.pdf)

![USB drive contents](usb-contents-screenshot.png)

## Key Sections

- **Contents** — identifies the mix of personal and work-related files (including PII) found on the drive
- **Attacker Mindset** — analyzes how an attacker could exploit the drive's contents against the owner or the organization
- **Risk Analysis** — outlines technical, operational, and managerial controls to mitigate USB baiting attacks, including port restrictions, antivirus scanning, and employee awareness training
