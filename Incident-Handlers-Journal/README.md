# Incident Handler's Journal

A running journal documenting incident investigations and cybersecurity tool usage throughout the Google Cybersecurity Certificate's detection and response course. Entries follow the 5 W's framework for incident analysis and are mapped to phases of the NIST Incident Response Lifecycle.

📄 [View the full journal](Incident%20handler%27s%20journal%20.pdf)

## Entries

- **Entry 1** — Ransomware incident at a healthcare clinic (Detection and Analysis)
- **Entry 2** — Malicious artifact analysis using VirusTotal and the Pyramid of Pain (Detection and Analysis)
- **Entry 3** — Phishing alert response using an incident response playbook (Containment, Eradication, and Recovery)
- **Entry 4** — Review of a data breach final report (Post-Incident Activity)
